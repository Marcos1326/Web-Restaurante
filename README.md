# Web Restaurante

Este proyecto es un ejemplo de una página web de un restaurante (invenado).

<hr>

## Informacion sobre la página

En esta pagina podremos observar 3 paginas html hipervinculadas. Cadauna de ella con su correspondida informacion.

Home / Carta / Contacto

En la página "Home" observaremos varias cosas:
<ul>
    <li>Informacion general
    <li>Imagenes del establecimiento
    <li>Horario del local
</ul>

En "Carta" observamos:

<ul>
    <li>Menu (Entrantes, plato principal, postre)
    <li>Bebidas
</ul>

"Contacto" contiene:

<ul>
    <li>Informacion (tlf., gmail, direccion...)
    <li>Mapa para tener una mejor visualizacion de la zona
    <li>Redes sociales del local
</ul>
